# CodeAlpha_music-player-interface
🎵 Music Player Interface
Built during my Internship at CodeAlpha
A sleek, modern Music Player UI built with pure HTML, CSS, and JavaScript — no frameworks, no libraries. Features a dark glassmorphism design with smooth animations and a fully interactive playlist.

✨ Features
🎨 Dark Glassmorphism UI — Premium dark theme with purple, cyan & pink accents
💿 Spinning Album Art — Vinyl-style rotating disc animation while playing
▶️ Playback Controls — Play/Pause, Previous, Next, Shuffle, Repeat
📊 Progress Bar — Clickable seek bar with real-time time display
🔊 Volume Control — Smooth slider with live percentage display
📋 Playlist Sidebar — Full track list with active track highlight
📡 Equalizer Animation — Animated EQ bars on the active playing track
🔁 Autoplay Toggle — Switch autoplay on/off with animated toggle
🌈 Animated Background — Subtle gradient hue-shift animation
📱 Responsive Design — Works across all screen sizes
🛠️ Built With
Technology	Purpose
HTML5	Structure & Markup
CSS3	Styling, Animations, Glassmorphism
JavaScript (Vanilla)	Interactivity & Logic
Google Fonts	Plus Jakarta Sans + Fira Code
⚡ Zero dependencies — runs directly in any browser.

🚀 Getting Started
Option 1 — Open directly
Just double-click index.html — it opens in any browser instantly.

Option 2 — Live Server (VS Code)
Install the Live Server extension in VS Code
Right-click index.html → Open with Live Server
🎮 How to Use
Action	How
Play / Pause	Click the ▶ button
Change Track	Click any song in the playlist
Seek	Click anywhere on the progress bar
Volume	Drag the volume slider
Shuffle	Click the shuffle icon
Repeat	Click the repeat icon
Autoplay	Toggle the Autoplay switch
